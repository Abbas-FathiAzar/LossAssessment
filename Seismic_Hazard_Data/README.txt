This folder contains data of seismic hazard curve.

The first column is spectral acceleration (in the unit of "g") and the second column is the mean annual frequency of exceedance.

"SeismicHazardData_3.660sec.txt" contains seismic hazard curve data for isolated building (lower bound isolator properties).
